# **App Name**: Nepali Dashboard

## Core Features:

- Current Date and Time: Display the current Nepali date, time and day.
- Weather Information: Show weather information for Kathmandu, including temperature, conditions, humidity and wind speed. Data will be automatically updated regularly.
- News Headlines: Display the latest news headlines from various Nepali sources. Data will be automatically updated regularly.
- Horoscope: Select your zodiac sign to view the daily horoscope. Data will be automatically updated regularly.
- Nepali Calendar Events: List upcoming Nepali festivals and holidays. Data will be automatically updated regularly.
- Music Player: Music player displaying FM stations. Data will be automatically updated regularly.
- Security Camera Feeds: Display live feeds from security cameras. Data will be automatically updated regularly.

## Style Guidelines:

- Primary color: Purple (#9370DB), transitioning to light purple/pink gradient, providing a sense of modernity and serenity.
- Background color: Very light grayish purple (#F5F5F5) for a clean and subtle backdrop.
- Accent color: Light blue (#ADD8E6) for interactive elements and highlights.
- Font: 'PT Sans', a humanist sans-serif, will be used for both headlines and body text to provide a balance of readability and a modern look.
- Tiles with rounded corners for displaying information in an organized way
- Use consistent and minimalist icons, filled, with rounded corners
- Gentle transitions for changes on screen
